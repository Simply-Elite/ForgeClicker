# ForgeClicker
An Actual Auto-Clicker made by CodeForge.

How to Download - 
1. Click on the latest release/ Version you'd like to download
2. You will be redirected to that release.
3. Click on the .zip file and your Download will start shortly.

Thanks for Using my app, make sure to leave a review and feedback, it matters a lot.
Updates will be released based on user feedback.
Consumer use only, reach the author for permission and other requests.
