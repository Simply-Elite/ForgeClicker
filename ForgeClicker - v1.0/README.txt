ForgeCLICKER V1.0

A lightweight Windows auto clicker built using Python - Flask, HTML, CSS and JavaScript.

FEATURES

• Adjustable CPS
• Configurable Start Delay
• Auto Stop Timer
• Emergency Stop (F8)
• Click Counter
• CPU Usage Monitor
• RAM Usage Monitor
• Modern Dashboard Interface

QUICK START

1. Launch AutoClicker.exe
2. Set your desired CPS.
3. Configure Start Delay if required.
4. Configure Auto Stop duration.
5. Press START.
6. Move your cursor to the target area.
7. Press F8 at any time to stop instantly.

SAFETY

• Keep CPS within reasonable limits.
• Very high CPS values may affect system responsiveness.
• Emergency Stop Key: F8
• Do not move the files, because they may cause errors in the program.

SYSTEM REQUIREMENTS

• Windows 10 or Windows 11
