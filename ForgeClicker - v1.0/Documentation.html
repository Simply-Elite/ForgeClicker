User Documentation

Introduction

AutoClicker is a lightweight Windows utility designed to automate left mouse clicks using a simple dashboard interface.

Features

  • Adjustable CPS (Clicks Per Second)
  • Start Delay
  • Auto Stop Timer
  • Emergency Stop (F8)
  • Click Counter
  • CPU Usage Monitor
  • RAM Usage Monitor

Quick Start

  1. Launch AutoClicker.exe
  2. Select a CPS value.
  3. Set Start Delay if required.
  4. Set Auto Stop duration.
  5. Press Start.
  6. Move your cursor to the target area.

Emergency Stop

Press F8 at any time to instantly stop the clicker.

Safety Guidelines

  • Use reasonable CPS values.
  • Higher CPS values may impact responsiveness.
  • Always keep the F8 hotkey in mind.
  • Test settings before long runs.
